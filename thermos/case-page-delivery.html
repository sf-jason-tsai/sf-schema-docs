1. 交付摘要

核心功能

  • Status Path（5 段可點擊）：新案件 → 案件分派 → 處理中 → 待客戶回覆 → 已結案。點擊即時透過 Apex + LDS 更新 Case.Status。
  • 案件詳情（inline 顯示）：類型/通路/優先度/保固/客戶/會員等級/LINE 綁定/詢問商品，帶客戶 90 天服務指標。
  • AI 客服助手（Einstein LLM 真接）：KM 相似度搜尋、AI 回覆草稿（可採用/重生）、下一步一鍵動作、加購商品推薦帶預估營收。
  • Composer（真寫 CaseComment）：案件回覆 / 內部備註切換，送出真寫入 CaseComment。
  • 處理時間軸：即時彙整 CaseComment + EmailMessage + CaseHistory 狀態變更 + Chatter，依日期分組。
  • 客戶其他案件：顯示該客戶歷史案件與平均結案時間。

驅動客戶購買 Agentforce 的設計亮點

  • Einstein Trust Layer 真串接：Hero 上有 Einstein · Live / Preview Mode badge，`isReal=true` 時直接顯示是真的在跑（我們 smoke test 驗證了 `isReal=true`）。
  • Credits 累計器：Hero 顯示 Agent credits {n}；每次 AI 動作即時 +N 顯示消耗，讓客戶感受「用一次收一次」的商業模式。
  • AI Deflection 時間預測：Hero 顯示「AI 估 12 分鐘結案 · 同類 22 分鐘」，量化 AI 加速效益。
  • Sentiment 客戶情緒：Hero 一目了然（生氣 / 困擾 / 平靜 / 滿意）— 客戶進頁 3 秒判斷風險。
  • 一鍵 Agent Actions：8 種預設動作（補發兌換碼、開 RMA、贈點道歉、加購推薦、物流通知、零件報價、推送 KM、升級 MIS），每個都標 credits 成本。
  • Cross-sell 營收預估：加購商品直接標「預估潛在營收 +NT$ 617」— 直接對應 Steven 說的「提升收入」。
