Service Cloud客戶服務雲

以 Case 為核心的全通路客服平台。整合 Email、電話、Live Chat、Messaging、社群、Field Service， 加上 Knowledge、Entitlement/SLA、Omni-Channel 派工，是 SFDC 最成熟的產品線。

Omni-ChannelPresence、Skill-based Routing、Supervisor
Field ServiceWork Order、Service Appointment、Resource Scheduling
Digital EngagementLive Chat、Messaging、Social、In-App
典型案件生命週期
Contact / Web Form → Case → Entitlement + Milestone → CaseComment / Knowledge → Work Order (可選) → Closed

Commerce Cloud電商雲 · B2B / B2C / D2C

原生於 Salesforce Platform 的電商站台（B2B Commerce / B2C Commerce on Core）。 資料模型與 CRM / Service 共用同一份 Account/Product/Order，是 Unified Commerce 的最大優勢。 注意：SFCC（B2C Commerce Cloud, 舊 Demandware）是另一套 SaaS 平台，資料模型不同。

Manufacturing Cloud製造雲
